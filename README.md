# 空调制冷系统仿真软件

一个专业的空调系统性能仿真平台，基于热力学物理模型，通过输入各部件参数，可以准确仿真系统的运行状态，包括温度、压力、制冷量、COP等关键性能指标。

## ✨ 功能特性

### 核心功能
- **稳态仿真计算**：基于物理模型的高精度稳态仿真
- **部件级建模**：压缩机、冷凝器、节流阀、蒸发器、风机等独立模型
- **多制冷剂支持**：支持R32、R410A、R22、R134a、R290等常用制冷剂
- **参数化分析**：灵活调整各部件参数，分析对系统性能的影响
- **可视化展示**：压焓图、性能参数图表、动态曲线展示
- **报告导出**：支持CSV格式的仿真结果导出

### 输入参数
#### 压缩机参数
- 排量、容积效率、等熵效率、电机效率
- 额定转速、制冷剂类型

#### 换热器参数
- **冷凝器**：换热面积、总换热系数、压力降、污垢系数
- **蒸发器**：换热面积、总换热系数、压力降、污垢系数

#### 节流阀参数
- 流量系数、阀门开度

#### 风机参数
- **冷凝器风机**：额定风量、额定风压、额定功率、效率
- **蒸发器风机**：额定风量、额定风压、额定功率、效率

#### 运行工况
- 环境温度、回风温度、相对湿度
- 压缩机运行转速

### 输出结果
#### 系统总性能
- 制冷量、总功耗、COP、EER

#### 压缩机参数
- 吸排气压力/温度、输入功率、质量流量、等熵效率

#### 冷凝器参数
- 冷凝温度/压力、过冷度、换热量、冷却介质出口温度

#### 蒸发器参数
- 蒸发温度/压力、过热度、制冷量、送风温度

#### 节流阀参数
- 进出口温度/压力、出口干度

#### 可视化
- 压焓图（p-h图）直观展示制冷循环过程

## 🚀 快速开始

### 环境要求
- Python 3.9+
- pip 包管理器

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行软件
```bash
streamlit run app.py
```

软件会自动在浏览器中打开，默认地址为 `http://localhost:8501`

## 📦 技术栈

- **后端计算**：Python + NumPy + SciPy + CoolProp
- **前端界面**：Streamlit + Plotly + Pandas
- **热力学物性库**：CoolProp (开源、高精度热力学物性计算库)

## 📖 使用说明

1. 在左侧边栏输入各部件的参数
2. 设置运行工况（环境温度、回风温度、湿度等）
3. 点击"运行仿真"按钮开始计算
4. 查看仿真结果，包括系统性能、各部件参数和压焓图
5. 可以将结果导出为CSV文件进行后续分析

## 🧮 物理模型说明

### 压缩机模型
采用半经验模型，考虑容积效率和等熵效率，准确计算压缩机的吸排气状态、质量流量和功耗。

### 换热器模型
采用分布参数法，考虑换热系数、压力降、两相流特性，分别计算冷凝器和蒸发器的换热性能。

### 节流阀模型
基于等焓节流过程，考虑流量系数和开度特性，计算节流后的温度、干度和质量流量。

### 风机模型
基于风机特性曲线，计算不同风量下的风压和功耗。

## 🔧 自定义开发

### 添加新的制冷剂
在 `app.py` 的制冷剂选择下拉框中添加对应的制冷剂名称即可，CoolProp库已经内置了几乎所有常用制冷剂的物性参数。

### 修改物理模型
在 `simulator_core.py` 中找到对应的部件模型类，可以根据需要修改计算逻辑，添加更复杂的模型。

### 扩展功能
可以添加动态仿真、故障仿真、多工况对比、参数优化等高级功能。
